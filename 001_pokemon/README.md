# 🐉 Pokémon Data Analysis with Python

## 📌 プロジェクト概要
ポケモンのデータセットを使用して、
Python（pandas）によるデータ分析の基礎を学ぶための個人学習プロジェクトです。

データの読み込み、基本情報の確認、集計、ランキング作成を通して  
**DataFrameの構造理解**を目的としています。

---

## 🛠 使用技術
- Python 3.12
- pandas
- matplotlib（一部可視化で使用）

---

## 📊 データセット
- Pokemon（CSV形式）
- 各ポケモンのタイプ、能力値、伝説区分などを含む

---

## 🔍 実装内容
- CSVデータの読み込み（`pd.read_csv`）
- データの基本情報確認（`head`, `tail`, `info`, `shape`）
- タイプ別ポケモン数の集計
- 攻撃力が高いポケモン TOP10 の抽出
- 各ポケモンの総合ステータス算出（`sum(axis=1)`）
